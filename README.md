## 👋 Introduction

Ever wanted to join your favorite private torrent tracker or forum but got tired of waiting for open registration? Well, this program will not do magic — but it will help you take the edge off that frustrating wait.

**Torrent Tracker Registration Monitor (TTRM)** is a desktop utility designed to automatically monitor and report the account registration availability across multiple private torrent trackers. Instead of manually navigating to dozens of sites to check if the registration gates are open, this tool handles the scraping and monitoring for you through a modular, scalable plugin system powered by Selenium WebDriver.

## 💡 Motivation

Gaining access to top-tier private trackers is often a waiting game of attrition. Open registration windows are incredibly brief, unannounced, and easily missed if you are not constantly refreshing the sign-up pages.

This project was built to automate the tedious process of manual checking. By building a robust plugin architecture, the community can easily write and integrate small scripts (plugins) to check the specific DOM elements of any tracker. The app centralizes all these checks into a clean UI, running quietly and alerting you the moment a registration gate opens.

##### ⚡ The Real Question
###### Why waste your time manually refreshing dozens of tracker sign-up pages every day when a lightweight desktop tool can quietly monitor them all for you?.

## 🤖 Features

📦 Plugin-Based Architecture

The program works through a plugin system.

Each plugin is composed of three main elements:

-  JSON file
	Contains the plugin's basic metadata:
	name, description, login or registration URL, etc.

- PNG, JPG, BMP, or ICO image file
	This is usually a banner or favicon
	that visually represents the plugin.
	A square image is recommended,
	preferably 48x48 pixels.

- VB file
	Contains the VB.NET source code that implements the 
	plugin's logic and behavior within the program.

📁 Plugins Location

All plugins are located in the `plugins` folder. If you don't find one of them useful and want to remove it from the program, you can delete the folder corresponding to the plugin name, or move it to another directory outside of its original location.

## 🚀 Getting Started (standard users)

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/Torrent-Tracker-Registration-Monitor/releases/latest)**.
2. Download the latest `.zip` archive or the `.exe` setup installer, depending on your preference.
3. If you downloaded the `.zip` archive, extract its contents to your preferred directory.
   <br> 
   If you downloaded the `.exe` file, run it and follow the installation wizard.
4. Run the executable file to launch the application.

💡 Remember to read the included `README.txt` file located in the `docs` folder for more details and also tips for using the application. 
   Or click [here](https://raw.githubusercontent.com/ElektroStudios/Torrent-Tracker-Registration-Monitor/main/Source/TTRM/docs/english/README.txt) to read the document now.
